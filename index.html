Tuner-G

Precision Guitar Tuning for iOS

Professional-grade chromatic tuner for guitar, bass, and ukulele. Accurate pitch detection with a clean, intuitive interface.

Features

Accurate Pitch Detection

Advanced YIN and PYIN algorithms provide professional-grade accuracy for reliable tuning across all frequency ranges.

Multiple Instruments & Tunings

Support for guitar, bass, and ukulele with standard, drop, and open tunings. Chromatic mode for any instrument.

Beautiful Interface

Clean neumorphic design with visual pitch gauge, waveform display, and clear tuning feedback. Dark and light modes included.

Customizable Settings

Adjust reference pitch (A4), sensitivity levels, detection algorithms, and choose between sharp/flat notation.

Reference Tone Generator

Play reference tones for any string to tune by ear. Long-press string buttons to hear the target pitch.

Support with Tips

Enjoy ad-free experience by supporting development with optional tips. One-time purchase removes all ads forever.
